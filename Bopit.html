Project Overview and Purpose 👻

This project, Bopit, was developed as part of the Computer Science and app development course. Its primary purpose is to show hoe to create a click app. The key challenges involved figuring where and when to put the specific code. Ultimately, this project serves as a functional example of learning to create code and different types of video games.

The technology stack includes: app inventor, html, javascript, and css
